# Code of Conduct

## Purpose
This Code of Conduct provides guidelines to ensure the deni-ai community remains a safe and welcoming space for everyone. We aim to cultivate an open and collaborative environment.

## Scope
This Code of Conduct applies to all community activities, including:
- Commits, reviews, issues, and discussions within the repository
- Online and offline events or chats
- Official social media and community communications

## Expected Behavior
- Treat each other with respect and courtesy
- Provide constructive feedback
- Welcome diverse backgrounds, experiences, and perspectives
- Support others and act collaboratively
- Use language appropriate for a public setting

## Unacceptable Behavior
- Discriminatory, insulting, or hostile remarks and harassment
- Sharing personal information without consent, doxxing
- Threats, stalking, or persistent harassment
- Unwanted sexual remarks or imagery
- Actions that disrupt community operations

## Enforcement
Project maintainers will respond to violations of this Code of Conduct in a reasonable and appropriate manner. When a violation is confirmed, one or more of the following actions may be taken:
- Request for correction
- Warning
- Temporary suspension
- Permanent suspension

## Reporting
If you experience or witness a violation or have concerns, please contact:
- security@deniai.app

Reports will be kept as confidential as possible, and retaliation is prohibited. We will investigate and respond appropriately based on the situation.

## Attribution
This Code of Conduct is based on the Contributor Covenant and adapted for deni-ai.
